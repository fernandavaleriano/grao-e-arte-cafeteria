Grão & Arte | Coffee Experience Landing Page
Landing page de alta fidelidade desenvolvida para uma cafeteria boutique, focada em experiência do usuário (UX) e conversão direta via canais digitais.

Link para Netlify  https://graoearte.netlify.app/


Quick Stack
Frontend: HTML5 Semântico, CSS3 (Flexbox/Animations).

Interatividade: JavaScript para controle de mídia e navegação.

Deploy/CI-CD: GitHub integrado ao Netlify.

Engenharia de Interface
Visualização de Produto 360º
Implementação de um componente de vídeo em loop infinito (autoplay, loop, muted) que demonstra a personalização física do produto. Esta técnica foi escolhida para aumentar o tempo de retenção do usuário na página e reduzir a fricção na percepção de valor do item personalizado.

Arquitetura de Navegação Smooth-Scroll
Utilização de IDs de âncora combinados com a propriedade scroll-behavior: smooth. Esta abordagem elimina saltos abruptos de tela, proporcionando uma transição elegante entre a narrativa da marca e o cardápio operacional.

Otimização de Conversão (Lead Gen)
Integração direta com a API do WhatsApp através de links parametrizados. O botão de ação principal (CTA) foi projetado para abrir em nova aba (target="_blank"), garantindo que o usuário não perca o contexto da navegação no site.

Design System
Tipografia e Cores: Paleta baseada em tons terrosos (#4B2C20) e fundos orgânicos (#F3F0EB) para evocar a estética do café artesanal.

Identidade Visual: Implementação de Favicon customizado para garantir reconhecimento da marca em múltiplas abas do navegador.

Como Executar o Projeto
Clone o repositório.

Abra o arquivo index.html em qualquer navegador moderno.

Para produção, o projeto está configurado para deploy automático via branch main.
